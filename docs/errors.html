Install & dependencies

Installer and host prerequisites

These errors indicate a missing package, binary, or an unsupported host configuration.

docker not found in PATH. Install Docker and retry

Docker is missing or not running. Install Docker Engine and ensure docker info works. Restart the Docker service and re-run the installer.

docker compose not available. Install Docker Compose v2 (docker compose) or docker-compose

Upgrade Docker or install the Compose plugin. Run docker compose version to verify.

docker-compose.yml not found from ... upward; run bootstrap from the repo root

Run gungnr bootstrap from the repo root so the CLI can find docker-compose.yml.

cloudflared not found in PATH. Install cloudflared and retry

Install cloudflared and confirm cloudflared --version succeeds.

Cloudflared & tunnels

Tunnel login, credentials, and runtime

Tunnel setup issues are usually caused by missing credentials, invalid config, or process failures.

cloudflared tunnel login failed

Re-run cloudflared tunnel login manually and complete the browser auth. Confirm the correct Cloudflare account is selected.

cloudflared credentials not found after login

The CLI expects ~/.cloudflared/cert.pem. Confirm the file exists and is readable. Use the same directory during login and bootstrap.

cloudflared tunnel credentials not found after create

The tunnel JSON should appear under ~/.cloudflared. Check permissions and retry tunnel creation.

cloudflared tunnel run failed

Check ~/.cloudflared/cloudflared.log for errors. Verify the tunnel ID exists and the config points to the correct credentials file.

tunnel did not report active connections before timeout

The tunnel process did not report connections. Run cloudflared tunnel info, verify outbound network access, and confirm the process is running.

DNS & domains

Cloudflare zone and DNS routing

These errors occur when the API token cannot read zones or update DNS records.

Cloudflare zone lookup failed / no active zone found

Ensure the domain is added to Cloudflare and the token has Zone read access. Use the root domain (example.com), not a subdomain.

Cloudflare DNS lookup failed / DNS record not found after routing

Confirm the token includes DNS edit permissions. Re-run cloudflared tunnel route dns <tunnel-uuid> panel.example.com and check the DNS record in Cloudflare.

Panel & auth

Panel startup and access

If the stack is running but the panel is unavailable or login loops, use these fixes.

timed out waiting for API health / health check returned 500

The stack started but the API did not become healthy. Check make logs, confirm Postgres is up, and ensure port 80 is free on the host.

template catalog not configured / template creation disabled

Configure the GitHub App in Host Settings. Template creation is disabled until App ID, client credentials, installation ID, and private key are set.

login loop or access denied after GitHub login

Confirm the user is on the allowlist. The bootstrap user is seeded as SuperUser, but additional users must be added in the panel.

API error codes

Standardized API errors

Every API error now includes a stable code. Search by code (for example AUTH-401) to jump directly to the recovery steps.

Error response shape: { code, message, error, docsUrl }. The UI surfaces the code in toast messages as [CODE] prefixes.

Core & validation

CORE-400Bad request

The request payload or query parameters are invalid. Re-check the request shape and field names.

CORE-404Resource not found

The referenced resource does not exist. Verify IDs, names, and that the resource still exists.

CORE-500Internal server error

An unexpected server error occurred. Check make logs and the API container logs for details.

VAL-400-FIELDSField validation failed

One or more fields failed validation. Inspect any fields map returned by the API for specifics.

VAL-400-NAMEInvalid project or service name

Names must be lowercase and use letters, numbers, or dashes. Trim spaces and avoid uppercase characters.

VAL-400-SUBDOMAINInvalid subdomain

Subdomains must be lowercase and use letters, numbers, or dashes. Avoid dots and underscores.

VAL-400-PORTInvalid port

Ports must be between 1 and 65535. Use a different port if the requested one is reserved or unavailable.

VAL-400-DOMAINInvalid domain

The provided domain is not a valid hostname. Use a full domain like example.com.

VAL-400-DOMAIN-MISSINGBase domain is not configured

Bootstrap should set the base domain. Verify your bootstrap-generated .env and the Host Settings domain value.

VAL-400-DOMAIN-NOT-CONFIGUREDDomain is not configured

The requested domain is not available to the panel. Choose the base domain or a Cloudflare zone visible to your token.

VAL-400-CONTAINER-NAMEInvalid container name

Container names must start with a letter or number and may include ., _, or -.

Authentication

AUTH-401Unauthenticated

Your session is missing or expired. Log in again via GitHub and retry the action.

AUTH-401-CREDENTIALSInvalid credentials

The admin test token credentials are incorrect. Verify ADMIN_LOGIN and ADMIN_PASSWORD in the environment.

AUTH-403Forbidden

The user is not allowed to access the resource. Confirm the user is on the allowlist with the correct role.

AUTH-403-ADMINAdmin role required

This action is restricted to Admin or SuperUser roles. Use an admin account or update the user role in Users.

AUTH-400-CALLBACKMissing OAuth callback data

The OAuth callback is missing code or state. Restart the login flow and avoid opening stale callback URLs.

AUTH-400-STATEInvalid OAuth state

The OAuth state cookie does not match the callback. Clear cookies for the panel domain and retry login.

AUTH-404-TEST-TOKENTest token endpoint disabled

The admin test-token endpoint is disabled unless ADMIN_LOGIN and ADMIN_PASSWORD are set.

AUTH-400-TEST-TOKENInvalid test token payload

The request body for /test-token is invalid. Send JSON with login and password fields.

AUTH-500-STATEFailed to generate OAuth state

The server could not generate a secure OAuth state. Check container entropy sources and restart the API.

AUTH-500-LOGINLogin failed

The OAuth exchange or GitHub lookup failed. Confirm GitHub OAuth credentials and callback URL are correct.

AUTH-500-SESSIONFailed to create session

The server could not encode the session. Verify SESSION_SECRET is set and stable across restarts.

Jobs

JOB-400-IDInvalid job id

The job id is not a valid positive number. Use the numeric job id from the Jobs view.

JOB-400-BODYInvalid job request body

The request body is malformed. Ensure JSON is valid and matches the endpoint schema.

JOB-404Job not found

The job no longer exists. Refresh the Jobs list and retry using an existing job id.

JOB-409-FINISHEDJob already finished

The job has already completed or failed. Use retry instead of stop, or create a new job.

JOB-409-RUNNINGJob already running

The job is currently running. Wait for it to finish or cancel it if the action supports stopping.

JOB-409-NOT-STOPPABLEJob is not stoppable

This job status cannot be stopped. Inspect the job status and logs to understand the current lifecycle stage.

JOB-409-NOT-RETRYABLEJob is not retryable

Only failed jobs can be retried. Wait for the job to fail, or rerun the action manually.

JOB-500-LISTFailed to load jobs

The server could not load jobs from the database. Check Postgres health and API logs.

JOB-500-STOPFailed to stop job

The API could not update the job state. Check database connectivity and retry.

JOB-500-RETRYFailed to retry job

The retry job could not be queued. Check the runner status and the API logs.

JOB-500-STREAMJob streaming unsupported

The current server or proxy does not support streaming. Use the Jobs detail view and refresh periodically.

Host & Docker

HOST-400-PROJECTInvalid project name

The project filter contains unsafe characters. Use only letters, numbers, dots, underscores, or dashes.

HOST-400-CONTAINERInvalid container input

The container name is missing or invalid. Choose a container from the list instead of typing a custom name.

HOST-400-BODYInvalid request body

The request body is malformed. Ensure the action payload is valid JSON.

HOST-403-ADMINAdmin role required

Docker mutations (stop, restart, remove) require Admin or SuperUser access.

HOST-500-DOCKERDocker command failed

The Docker command failed on the host. Check that Docker is running and that the API has socket access.

HOST-500-USAGEFailed to load Docker usage

Docker usage inspection failed. Run docker system df on the host to verify Docker health.

HOST-500-LOGSFailed to stream Docker logs

Log streaming failed. Confirm the container exists and that Docker logs are accessible on the host.

HOST-500-STREAMStreaming unsupported

The server or proxy does not support streaming responses. Try refreshing the view or check logs directly.

Settings

SETTINGS-500-LOADFailed to load settings

The API could not load settings from storage. Check database connectivity and API logs.

SETTINGS-500-SOURCESFailed to resolve settings sources

The API could not resolve config sources. Verify the environment and database are consistent.

SETTINGS-403-ADMINAdmin role required

Updating settings requires Admin or SuperUser access.

SETTINGS-400-BODYInvalid settings payload

The settings request body is invalid. Confirm it matches the settings schema and uses correct field names.

SETTINGS-500-UPDATEFailed to update settings

The API could not persist settings. Check database health and retry the update.

SETTINGS-400-PREVIEWCloudflared preview failed

The configured cloudflared config path could not be read. Verify the path exists and is readable by the API container.

SETTINGS-500-SYNCFailed to sync settings

Syncing settings from the environment failed. Check the Cloudflare-related environment variables and retry.

Cloudflare & GitHub

CF-500-SERVICECloudflare service unavailable

The Cloudflare integration is not initialized. Restart the API and confirm Cloudflare settings are present.

CF-502-PREFLIGHTCloudflare preflight failed

The Cloudflare preflight check failed. Confirm the token, zone, and tunnel settings are correct.

CF-502-ZONESCloudflare zone lookup failed

Fetching zones from Cloudflare failed. Check the API token scopes and account visibility.

CF-400-TOKENCloudflare token missing

The Cloudflare API token is required. Set it during bootstrap or update it in Host Settings.

CF-400-ACCOUNTCloudflare account ID missing

The Cloudflare account ID is required for certain API calls. Confirm it is present in the environment or settings.

CF-400-ZONECloudflare zone ID missing

The Cloudflare zone ID is missing. Ensure the selected domain is backed by a visible Cloudflare zone.

CF-400-TUNNELCloudflared tunnel reference missing

The tunnel name or ID is required. Re-run bootstrap tunnel setup or verify the tunnel reference in settings.

CF-409-TUNNEL-LOCALTunnel is locally managed

The tunnel is configured locally and cannot be updated via the Cloudflare API. Use local config updates instead.

GH-500-SERVICEGitHub service unavailable

The GitHub integration is not initialized. Restart the API and confirm GitHub settings are configured.

GH-500-CATALOGGitHub catalog failed

The template catalog could not be loaded. Confirm GitHub App settings are correct and the installation has access.

GH-400-TOKENGitHub token missing

A GitHub installation token is required for this action. Configure the GitHub App and installation in Host Settings.

Users & roles

USER-400-IDInvalid user id

The user id must be a positive integer. Use the user id returned by the API or shown in the Users view.

USER-400-PAYLOADInvalid user payload

The user request body is invalid. Ensure the payload is valid JSON with the expected fields.

USER-400-ROLEInvalid role

Roles must be admin or user when updating roles through the API.

USER-400-LOGINLogin is required

A GitHub login is required when adding a user to the allowlist.

USER-404User not found

The user does not exist. Refresh the Users list and retry with a valid user id.

USER-404-GITHUBGitHub user not found

The provided GitHub username could not be resolved. Check spelling and that the user exists publicly.

USER-400-LAST-SUPERUSERCannot demote last superuser

At least one SuperUser must exist. Promote another user before demoting the current SuperUser.

USER-400-SUPERUSERCannot remove superuser

SuperUsers cannot be removed through the allowlist delete action.

USER-500-LISTFailed to load users

The API could not load users from storage. Check database health and API logs.

USER-500-UPDATEFailed to update user

The user update failed to persist. Check database connectivity and retry the update.

USER-500-CREATEFailed to create user

The user could not be created. Check GitHub lookup success and database health.

USER-500-DELETEFailed to delete user

The user could not be deleted. Check database health and retry.

Projects

PROJECT-500-LISTFailed to load projects

The API could not load projects. Check database connectivity and API logs.

PROJECT-500-LOCALFailed to load local projects

Local template discovery failed. Verify TEMPLATES_DIR is set and readable by the API container.

PROJECT-400-BODYInvalid project request body

The project request body is invalid. Confirm field names and types match the API schema.

PROJECT-403-ADMINAdmin role required

Project mutations require Admin or SuperUser access.

PROJECT-400-CREATECreate-from-template validation failed

The create-from-template request failed validation. Check name, subdomain, domain, and port inputs.

PROJECT-400-DEPLOYDeploy-existing validation failed

The deploy-existing request failed validation. Confirm the service exists and the port/domain inputs are correct.

PROJECT-400-FORWARDForward-local validation failed

The forward-local request failed validation. Re-check the service name, subdomain, domain, and port.

PROJECT-400-QUICKQuick service validation failed

The quick service request failed validation. Confirm subdomain, domain, and port values are valid.

PROJECT-400-TEMPLATETemplate selection invalid

The template reference is invalid or not in the allowlist. Select a template from the catalog instead of typing one.

Audit & RBAC

AUDIT-500-LISTFailed to load audit logs

The audit log query failed. Check database health and API logs.

RBAC-409-SUPERUSER-CAPSuperUser cap exceeded

More than two SuperUsers were detected and pruned during seeding. Review the Users list to confirm the final state.
