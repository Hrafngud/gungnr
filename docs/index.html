Gungnr

Hit the clouds from home.

Container-based deployment orchestration that turns any internet-connected computer into a server. Built on Docker (compose + orchestration), Cloudflare (networking), and GitHub (auth + templates) to move from development to production fast. The goal is simple: use free, modern tools to make home servers easy and ship small projects to the web with speed.
